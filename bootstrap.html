Project Detail + Quick Links

Main content takes 8 columns on large screens; sidebar takes 4. Both stack on mobile.

StatScout

StatScout is a full-stack multi-sport analytics platform that delivers real-time betting odds, hit rates, and custom trust scores for NBA player props and soccer markets. Built with React on the frontend and Flask on the backend, with a PostgreSQL database hosted on Supabase. The live projection engine takes a player's first-half stats and estimates their full-game total using weighted historical averages.

Tech stack: React, Flask, PostgreSQL, Supabase, Vercel, Render.
